## 2. Typography

### Canva Font Pairings

**Option A (Recommended):**
- **Headlines:** Playfair Display (Bold)
- **Subheads:** Montserrat (SemiBold)
- **Body:** Montserrat (Regular)

**Option B (Modern Alternative):**
- **Headlines:** DM Serif Display
- **Subheads:** Inter (SemiBold)
- **Body:** Inter (Regular)

**Option C (Approachable):**
- **Headlines:** Lora (Bold)
- **Subheads:** Source Sans Pro (SemiBold)
- **Body:** Source Sans Pro (Regular)

### Font Sizes by Platform

| Element | Facebook | Instagram | LinkedIn |
|---------|----------|-----------|----------|
| Main headline | 48-64px | 40-56px | 36-48px |
| Subhead | 28-36px | 24-32px | 24-28px |
| Body text | 18-24px | 16-20px | 16-20px |
| Caption/fine print | 14-16px | 12-14px | 14-16px |

### Text Styling Rules

- **Line height:** 1.4-1.6 for body text
- **Letter spacing:** 0 for body, +2% for headlines
- **Maximum line length:** 45-65 characters
- **Alignment:** Left-aligned (never centered body text)
- **ALL CAPS:** Headlines only, never body text

## 4. Visual Style Guidelines

### Photography Style

**DO use images that are:**
- Natural, candid moments (not staged)
- Soft, natural lighting (golden hour preferred)
- Australian landscapes and wildlife
- Small groups in nature (2-6 people)
- Real birding scenarios (binoculars, field guides, patience)
- Native Australian birds in natural habitats

**DON'T use images that are:**
- Overly saturated or HDR-processed
- Generic stock "happy tourist" poses
- Non-Australian wildlife
- Large crowds
- Urban settings (unless relevant)
- Anything that feels like marketing stock

### Stock Image Search Terms

**Australian Birds:**
- "Australian kingfisher"
- "Superb fairy wren"
- "Rainbow lorikeet wild"
- "Wedge-tailed eagle"
- "Australian pelican"
- "Kookaburra eucalyptus"
- "Cockatoo Australia"
- "Cassowary Queensland"

**Australian Landscapes:**
- "Australian bushland sunrise"
- "Eucalyptus forest"
- "Outback dawn"
- "Australian wetlands"
- "Kakadu landscape"
- "Great Ocean Road nature"
- "Daintree rainforest"
- "Australian national park"

**Birding Activity:**
- "Birdwatching binoculars"
- "Nature guide pointing"
- "Small group wildlife tour"
- "Field guide birds"
- "Wildlife photography Australia"

### Stock Image Sources

**Free:**
- Unsplash (search "Australia birds", "Australian wildlife")
- Pexels (good for landscapes)
- Pixabay (variable quality, check carefully)

**Paid (Higher Quality):**
- iStock (best Australian bird selection)
- Adobe Stock
- Shutterstock
- Getty Images

**Specialist (Recommended):**
- Australian Geographic image library
- BirdLife Australia (may require permission)
- eBird Macaulay Library (Creative Commons)

### Image Treatment

**Standard overlay for text readability:**
- Gradient overlay: `#1B4D3E` at 40-60% opacity
- Position: Bottom-up gradient
- Alternative: Dark vignette around edges

**Color grading direction:**
- Slightly desaturated (not Instagram-filter vibrant)
- Warm shadows
- Cool highlights
- Natural greens (not neon)

## 5. Logo Usage

### Primary Logo
*(To be designed - placeholder guidelines)*

**Clear space:** Minimum 20% of logo width on all sides
**Minimum size:**
- Digital: 120px wide
- Print: 25mm wide

### Logo Variations Needed

1. **Primary** - Full logo (icon + wordmark)
2. **Compact** - Icon only (for profile photos)
3. **Wordmark** - Text only (for tight spaces)
4. **Reversed** - White version for dark backgrounds

### Logo Don'ts
- Don't stretch or distort
- Don't add effects (shadows, glows)
- Don't change colors outside palette
- Don't place on busy backgrounds
- Don't rotate

## 6. Graphic Elements

### Icon Style
- Line weight: 2px stroke
- Style: Rounded corners, friendly but professional
- Color: Single color from palette (Deep Eucalyptus or Outback Gold)

### Borders & Containers
- Border radius: 8-12px (slightly rounded, not pill-shaped)
- Border width: 1-2px
- Border color: Mist (`#DFE6E9`) or Bushland Sage (`#6B8E7D`)

### Dividers
- Weight: 1px
- Color: Mist (`#DFE6E9`)
- Style: Solid (no dashes)

### Quote Styling
- Left border: 4px Deep Eucalyptus
- Background: Sky White with 50% opacity
- Font: Playfair Display Italic

### Number/Stat Callouts
- Large number: Playfair Display, 72px, Outback Gold
- Label: Montserrat SemiBold, 16px, Charcoal
- Container: Light background with subtle border

## 8. Tone of Voice (Visual Translation)

### Trust = Clean, Uncluttered
- Maximum 3 elements per composition
- Generous white space (40%+ of canvas)
- No gradients on text
- No drop shadows on UI elements

### Education = Clear Hierarchy
- One dominant element per design
- Clear reading order (top to bottom, left to right)
- Numbers and bullets for lists
- Progress indicators on carousels

### Authenticity = Natural, Not Polished
- Slightly imperfect photo crops are OK
- Avoid overly symmetrical layouts
- Let the bird/nature be the hero
- No decorative flourishes

### No Hype = Restraint
- Never use: "Amazing!", "Don't miss out!", "Limited time!"
- Maximum one exclamation mark per post
- Prefer periods and questions
- Let facts speak, not adjectives
