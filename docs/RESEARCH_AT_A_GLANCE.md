## Red Flags & Mitigations

### Risk 1: Facebook Backlash
```
RISK: Posts get deleted, account flagged for promotional content
MITIGATION: 70/30 rule, follow group rules, comment authentically first
```

### Risk 2: Content Quality Issues
```
RISK: Bird ID mistakes undermine credibility
MITIGATION: Partner with ornithology expert, cite sources, enable corrections
```

### Risk 3: Engagement Plateau
```
RISK: Organic growth hits ceiling at 2-3 months
MITIGATION: Budget for paid social testing ($1,500-2,500)
```

### Risk 4: Operator Resistance
```
RISK: Established operators view Quorum as threat
MITIGATION: Position as partnership, not replacement; early adopter benefits
```

### Risk 5: Seasonal Volatility
```
RISK: Australian spring (Sep-Nov) is peak season; other months slow
MITIGATION: Build 8-week content buffer; target seasonal tour types
```
